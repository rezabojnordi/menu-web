# Cafe Menu Project

این پروژه یک منوی کافی‌شاپ HTML/CSS/JS است.

## فایل‌ها
- `index.html` صفحه اصلی منو
- `style.css` استایل و ظاهر منو
- `script.js` فیلتر دسته‌بندی‌ها با کلیک
- `assets/` پوشه آماده برای عکس‌های اختصاصی شما

## روش اجرا
فایل `index.html` را داخل مرورگر باز کنید.

## تغییر عکس‌ها
در فایل `index.html` مقدار `src` عکس هر محصول را عوض کنید.

## تغییر قیمت‌ها
در فایل `index.html` مقدار داخل کلاس `price` را تغییر دهید.
مثال:
```html
<span class="price">₺120</span>
```
